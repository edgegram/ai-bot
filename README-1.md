# AI Companion Bot (Gemini + edge-tts, бесплатный стек)

Telegram-бот для общения с настраиваемым вымышленным персонажем.

- **Gemini API** — бесплатный LLM от Google для диалогов
- **edge-tts** — бесплатная озвучка, ключ не нужен

## Переменные окружения для Railway

- `BOT_TOKEN` — токен от @BotFather
- `GEMINI_API_KEY` — ключ с aistudio.google.com (тот, что ты уже получил)

## Start Command (то же самое, без изменений)

```
pip install -r requirements.txt && python bot.py
```

## Как работает бот

1. `/start` — мастер настройки: имя → возраст → тип → характер → голос → аватарка
2. Дальше просто пишешь текст — бот отвечает текстом и голосовым
3. `/profile` — посмотреть настройки
4. `/reset` — настроить заново

## Лимиты бесплатного тарифа Gemini

На модели `gemini-2.5-flash` бесплатный тариф даёт около 10 запросов в
минуту и 250 запросов в день (лимиты периодически меняются у Google,
это ориентир). Для личного бота обычно с запасом хватает.
